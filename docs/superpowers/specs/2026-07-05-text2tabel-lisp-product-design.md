# Text2Tabel AutoCAD LISP Product Design

## Status

Approved for implementation on 2026-07-05.

## Goal

Publish `Text2Tabel.lsp` on IVapps.pro by following the existing AutoCAD LISP product pattern used for `XRef2Current`.

## Catalog Entry

- Add a calculator catalog card with slug `text2tabel`.
- Place it in the existing `autocad-lisp` category.
- Use `displayMode: "product"` and `openUrl: "/products/text2tabel"`.
- Use the card label `Сторінка LISP`, icon `Code2`, standard `AutoCAD for Windows`, and a later order than `xref-to-current`.

## Product Page

- Add a product page at `/products/text2tabel`.
- Use type label `AutoCAD LISP`.
- Present `Text2Tabel` as an AutoCAD utility that creates native AutoCAD tables from selected text, line-drawn table grids, or one-by-one picked text values.
- Show facts for command `Text2Tabel`, version `1.0`, platform `AutoCAD for Windows`, and unsupported AutoCAD for Mac / AutoCAD LT.
- Describe the four modes: `ByLines`, `ByText`, `OneByOne`, and `TextToCell`.
- Include a warning that `TextToCell` writes selected text into an existing table cell and deletes the source text object after a successful write.
- Reuse the existing APPLOAD installation pattern.
- Reuse the proprietary free-use license wording style from the existing AutoCAD LISP page.

## Download

- Publish the source as `public/downloads/autocad-lisp/Text2Tabel.lsp`.
- Prepend the same style of proprietary header used by `XRef2Current.lsp`.
- Keep the original supplied program source unchanged after the header.

## Artwork

- Add a catalog image asset for `text2tabel`, generated by the existing icon generator.
- Use the existing AutoCAD palette and a compact `T2T` badge.

## Tests
