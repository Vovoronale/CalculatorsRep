# Residential Yard Areas Report Contract

Date: 2026-06-20
Calculator: `residential-yard-areas`
Status: Agreed source of truth for report text and formulas

2026-07-15 amendment status: Pending written-spec confirmation

This contract contains the agreed Ukrainian UI labels and the exact report
captions, display conditions, formulas, results, warnings, and errors. After
confirmation, it becomes the sole source of truth for implementation and tests.

## Report Title

```text
Розрахунок площ майданчиків у складі прибудинкової території
```

The same single title is used in the UI and DOCX. No additional report heading
is rendered.

## Input Changes

Replace the former greenery-sufficiency checkbox with:

```text
Label: Фактична площа зелених насаджень обмеженого користування, м²
Symbol: S_(озел,факт)
Type: number
Unit: м²
Default: empty
Display condition: Норматив площі фізкультурних майданчиків = Зменшений норматив
Constraint: after entry, S_(озел,факт) >= 0
```

Description:

```text
Введіть фактичну площу зелених насаджень обмеженого користування за проєктною документацією. Значення порівнюється з мінімальною площею 6 м² на одного мешканця та визначає можливість застосування зменшеного нормативу фізкультурних майданчиків. Джерело: примітка * до таблиці 6.4 ДБН Б.2.2-12:2019.
```

The separate-zone checkbox remains conditional for the reduced mode. Its formal
affirmative report text is:

```text
Окрему озеленену зону з фізкультурними майданчиками передбачено проєктом.
```

Use this apartment label everywhere:

```text
Кількість квартир із двома і більше кімнатами
```

For vacuum collection, use:

```text
Площу визначено відповідно до містобудівних і технічних умов.
```

## Calculation Rules

For every platform with parallel person and apartment rates, show:

```text
За розрахункову приймається більша з площ, визначених за кількістю мешканців і кількістю квартир згідно з п. 6.1.21 та таблицею 6.4 ДБН Б.2.2-12:2019.
```

Use `розрахункова площа`, never `прийнята площа`. Every symbol uses an
explicit parenthesized index: `N_(осіб)`, `N_(кв)`, `N_(1)`, `N_(2+)`,
`N_(гост)`, `S_(діт)`, `S_(відп)`, `S_(фіз)`, `S_(гост)`, `S_(вел)`,
`S_(відх)`, `S_(твар)`, `S_(госп)`, `S_(прибуд)`, `S_(озел,мін)`,
`S_(озел,факт)`, and the corresponding `q_(...)` symbols.

The default physical-culture mode remains the full norm. When the reduced mode
is selected, it applies only if the separate landscaped zone is provided and
`S_(озел,факт) >= S_(озел,мін)`. Otherwise the full rates are used
automatically, the applicable error is shown, and the report includes:

```text
Умови застосування зменшеного нормативу не виконано; розрахунок виконано за повним нормативом.
```

## Step 4: Фізкультурні майданчики

Caption:

```text
Визначення площі майданчиків для занять фізкультурою (п. 6.1.21, таблиця 6.4 та примітка * ДБН Б.2.2-12:2019):
```

When the reduced mode is selected, first validate that
`S_(озел,факт)` is a finite value not less than zero. A missing, non-finite, or
negative value is a blocking input error and follows the blocking-invalid rule;
do not render this step or apply automatic fallback.

For a valid finite `S_(озел,факт)`, show the greenery check before the
physical-culture formulas:

```text
S_(озел,мін) = 6 × N_(осіб) = 6 × <N_(осіб)> = <результат> м²
Фактична площа зелених насаджень обмеженого користування: S_(озел,факт) = <значення> м²
S_(озел,факт) ≥ S_(озел,мін): <фактична площа> ≥ <мінімальна площа> — умову виконано
```

or:

```text
S_(озел,факт) ≥ S_(озел,мін): <фактична площа> ≥ <мінімальна площа> — умову не виконано
```

If both reduced-mode conditions pass, show:

```text
q_(фіз,осіб) = 0,2 м²/особу
q_(фіз,кв) = 0,5 м²/квартиру
```

For the selected full mode, or after the agreed automatic fallback caused only
by a missing separate landscaped zone or a valid finite greenery area below
`S_(озел,мін)`, show:

```text
q_(фіз,осіб) = 2,0 м²/особу
q_(фіз,кв) = 5,0 м²/квартиру
```

Formulas:

```text
S_(фіз,осіб) = q_(фіз,осіб) × N_(осіб) = <чинний коефіцієнт> × <N_(осіб)> = <результат> м²
S_(фіз,кв) = q_(фіз,кв) × N_(кв) = <чинний коефіцієнт> × <N_(кв)> = <результат> м²
S_(фіз) = max(S_(фіз,осіб); S_(фіз,кв)) = max(<S_(фіз,осіб)>; <S_(фіз,кв)>) = <результат> м²
Розрахункова площа фізкультурних майданчиків: S_(фіз) = <результат> м²
Розрахункова база: <кількість мешканців / кількість квартир / обидві бази дають однакову площу>
За розрахункову приймається більша з площ, визначених за кількістю мешканців і кількістю квартир згідно з п. 6.1.21 та таблицею 6.4 ДБН Б.2.2-12:2019.
```

Errors:

```text
Зменшений норматив не можна застосувати: окрему озеленену зону з фізкультурними майданчиками не передбачено проєктом.
Зменшений норматив не можна застосувати: фактична площа зелених насаджень обмеженого користування має бути не меншою за <S_(озел,мін)> м².
```
