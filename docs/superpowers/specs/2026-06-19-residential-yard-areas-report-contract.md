# Residential Yard Areas Report Contract

Date: 2026-06-20
Calculator: `residential-yard-areas`
Status: Draft pending confirmation

This contract contains the agreed Ukrainian UI labels and the exact report
captions, display conditions, formulas, results, warnings, and errors. After
confirmation, it becomes the sole source of truth for implementation and tests.

## Report Title

```text
Розрахунок площ майданчиків у складі прибудинкової території
```

The same single title is used in the UI and DOCX. No additional report heading
is rendered.

## Input Changes

Replace the former greenery-sufficiency checkbox with:

```text
Label: Фактична площа зелених насаджень обмеженого користування, м²
Symbol: Sозел,факт
Type: number
Unit: м²
Display condition: Норматив площі фізкультурних майданчиків = Зменшений норматив
Constraint: Sозел,факт >= 0
```

Description:

```text
Введіть фактичну площу зелених насаджень обмеженого користування за проєктною документацією. Значення порівнюється з мінімальною площею 6 м² на одного мешканця та визначає можливість застосування зменшеного нормативу фізкультурних майданчиків. Джерело: примітка * до таблиці 6.4 ДБН Б.2.2-12:2019.
```

The separate-zone checkbox remains conditional for the reduced mode. Its formal
affirmative report text is:

```text
Окрему озеленену зону з фізкультурними майданчиками передбачено проєктом.
```

Use this apartment label everywhere:

```text
Кількість квартир із двома і більше кімнатами
```

For vacuum collection, use:

```text
Площу визначено відповідно до містобудівних і технічних умов.
```

## Calculation Rules

For every platform with parallel person and apartment rates, show:

```text
За розрахункову приймається більша з площ, визначених за кількістю мешканців і кількістю квартир згідно з п. 6.1.21 та таблицею 6.4 ДБН Б.2.2-12:2019.
```

Use `розрахункова площа`, never `прийнята площа`. Use indexed symbols,
including `Nосіб`, `Nкв`, `Sдіт`, `Sвідп`, `Sфіз`, `Sозел,мін`,
`Sозел,факт`, `Sприбуд`, and `Sтвар`.

The default physical-culture mode remains the full norm. When the reduced mode
is selected, it applies only if the separate landscaped zone is provided and
`Sозел,факт >= Sозел,мін`. Otherwise the full rates are used automatically and
the applicable error is shown.

Caption:

```text
Визначення площі майданчиків для занять фізкультурою (п. 6.1.21, таблиця 6.4 та примітка * ДБН Б.2.2-12:2019):
```

When the reduced mode is selected, always show the greenery check before the
physical-culture formulas:

```text
Sозел,мін = 6 × Nосіб = 6 × <Nосіб> = <результат> м²
Sозел,факт = <значення> м²
Sозел,факт ≥ Sозел,мін: <фактична площа> ≥ <мінімальна площа> — умову виконано
```

or:

```text
Sозел,факт ≥ Sозел,мін: <фактична площа> ≥ <мінімальна площа> — умову не виконано
```

If both reduced-mode conditions pass, use `0,2 м²/особу` and
`0,5 м²/квартиру`. Otherwise state that the full norm is applied and use
`2,0 м²/особу` and `5,0 м²/квартиру`.

Formulas:

```text
Sфіз,ос = qфіз,ос × Nосіб = <чинний коефіцієнт> × <Nосіб> = <результат> м²
Sфіз,кв = qфіз,кв × Nкв = <чинний коефіцієнт> × <Nкв> = <результат> м²
Sфіз = max(Sфіз,ос; Sфіз,кв) = max(<Sфіз,ос>; <Sфіз,кв>) = <результат> м²
```

Errors:

```text
Зменшений норматив не можна застосувати: окрему озеленену зону з фізкультурними майданчиками не передбачено проєктом.
Зменшений норматив не можна застосувати: фактична площа зелених насаджень обмеженого користування має бути не меншою за <Sозел,мін> м².
```

## Step 12: Умови застосування результатів

Caption:

```text
Умови застосування результатів:
```

All limitations appear once, only here:

```text
Сумарна площа Sприбуд не є мінімальною площею земельної ділянки житлового будинку за таблицею 6.3 ДБН Б.2.2-12:2019 і не включає площу забудови, проїздів, тротуарів, озеленення та інших елементів території.
Майданчики для вигулу домашніх тварин розміщують поза межами прибудинкової території на відстані не менше 40 м від вікон житлових будинків, майданчиків для ігор і відпочинку та фізкультурних майданчиків.
Для наземного способу збирання побутових відходів відстань до вікон житлових і громадських будівель та до дитячих, фізкультурних і рекреаційних майданчиків має бути не менше 20 м, а пішохідна доступність — не більше 100 м. Для підземного і вакуумного способів відстані визначаються технічними умовами.
Відстань від господарських майданчиків до найбільш віддаленого входу в житловий будинок має бути не більше 100 м.
Перевірка фактичного розміщення, під’їздів і нормативних відстаней не входить до цього розрахунку.
```
