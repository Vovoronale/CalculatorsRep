# Construction and Urban Planning Icons Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the 14 approved construction and urban-planning catalog illustrations with crisp 512 px PNG icons while retaining SVG artwork for every other calculator.

**Architecture:** Add one small path resolver that chooses PNG only for the approved slug set, keeping the component free of category logic. Generate the PNG files deterministically from focused SVG artwork so silhouettes and mathematical labels remain exact, then rasterize them with the bundled workspace image tooling.

**Tech Stack:** Next.js, TypeScript, Vitest, SVG source markup, Sharp rasterization, Browser plugin.
