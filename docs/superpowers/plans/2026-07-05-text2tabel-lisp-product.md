# Text2Tabel AutoCAD LISP Product Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish `Text2Tabel.lsp` as a downloadable AutoCAD LISP product page and catalog card.

**Architecture:** Reuse the existing product-mode calculator pattern. Store user-facing content in `data/content.json`, the downloadable source in `public/downloads/autocad-lisp/`, and the catalog artwork through `scripts/generate-construction-urban-icons.mjs`.

**Tech Stack:** Next.js App Router, TypeScript JSON loaders, Vitest, Sharp icon generation.

## Global Constraints

- Keep visible content JSON-driven through `data/content.json`.
- Use `displayMode: "product"` for the catalog card.
- The product route is `/products/text2tabel`.
- The download URL is `/downloads/autocad-lisp/Text2Tabel.lsp`.
- The supplied LISP program source remains unchanged after the added license header.
